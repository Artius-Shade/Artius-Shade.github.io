Introduction

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small- and large-scale projects.

Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

Core philosophy of Python is summarized in the document The Zen of Python, by Tim Peters, which includes aphorisms such as:

  • Beautiful is better than ugly.
  • Explicit is better than implicit.
  • Simple is better than complex.
  • Complex is better than complicated.
  • Readability counts.

Besides being powerful language with many awesome features, Python is easy to learn and highly beginner friendly and consistently ranks as one of the most popular programming languages.

History of Python

Python was conceived in the late 1980s by Guido van Rossum at Centrum Wiskunde & Informatica (CWI) in the Netherlands as a successor to the ABC programming language, which was inspired by SETL, capable of exception handling and interfacing with the Amoeba operating system. Its implementation began in December 1989. Van Rossum shouldered sole responsibility for the project, as the lead developer, until 12 July 2018, when he announced his "permanent vacation" from his responsibilities as Python's "benevolent dictator for life", a title the Python community bestowed upon him to reflect his long-term commitment as the project's chief decision-maker. In January 2019, active Python core developers elected a five-member Steering Council to lead the project.

Python 2.0 was released on 16 October 2000, with many major new features. Python 3.0, released on 3 December 2008, with many of its major features backported to Python 2.6.x and 2.7.x. Releases of Python 3 include the 2to3 utility, which automates the translation of Python 2 code to Python 3.

Python 2.7's end-of-life was initially set for 2015, then postponed to 2020 out of concern that a large body of existing code could not easily be forward-ported to Python 3. No further security patches or other improvements will be released for it. With Python 2's end-of-life, only Python 3.6.x and later are supported.

Python 3.9.2 and 3.8.8 were expedited as all versions of Python (including 2.7) had security issues leading to possible remote code execution[55] and web cache poisoning.

Conditional Execution

The simplest form of conditional statement in Python is the if statement.
>>> if 2 + 2 == 4:
...     print("I'm a math genius")

For double-branched conditional statement we can use the if else statement. In this type of conditional, one branch always gets executed.
x = int(input("Enter a number: ")) if x % 2 == 0:
    print("Even number found")
else:
    print("Odd number found")

For multi-branched conditional statement we can use the if elif else statement. elif is shorthand for "else-if", the else statement is optional in multi-branched conditional and can be ommited.
text = input("Enter your message: ")
if text == "hello":
    print("Hello :)")
elif text == "how are you?":
    print("I'm fine, how are you?")
elif text.startswith("bye"):
    print("Byee, have a nice day!")

Iteration

Iteration is an essential and commonly used thing in programming. Python has two ways to iterate over a series

  1. while loop

    We can write a while loop by using while keyword followed by an boolean expression. The loop keep going until the boolean expression evaluates to false.
    >>> n = 0
    >>> while n < 3:
    ...     print("spam")
    ...     n += 1
    ...
    spam
    spam
    spam

  2. for loop

    We can write a for loop by using for keyword followed by an iteration variable, in keyword and a series. The loop keep going until the series is exhausted.
    >>> items = ["chicken", 9999, True]
    >>> for item in items:
    ...     print(item)
    ...
    chicken
    9999
    True

Object Oriented Programmming

To create classes in Python we can use the class keyword followed by the class name and colon. Classes can have attributes and methods within them, once we define a class we can use it as an template to create a object, which is an instance of a class.
class Pet:
    hungry = True
    def __init__(self, name):
        self.name = name

    def feed(self):
        Pet.hungry = False
        return f"{self.name} has eaten"

    def play(self):
        Pet.hungry = True
        return f"{self.name} had lots of fun with you :)"
